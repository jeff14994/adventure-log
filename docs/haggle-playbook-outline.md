## 賣點一句話
> 「我在美國車行，用三句話砍掉六千美金。這本手冊教你一模一樣的流程。」

## 章節架構

### 0. 開場：$6000 是怎麼砍下來的（鉤子）
- 濃縮你的真實故事，讓讀者相信「這是實戰、不是理論」。

### 1. 開戰前的功課（Leverage 從這裡來）
- 查 TrueCar / Edmunds / KBB 成交價，抓「別人實際成交」數字
- Out-the-door price（含稅費）才是真價格，破解月付話術
- 貸款利率先自己銀行預核，別讓車行賺利差
- 同款車鎖定 3~5 家，製造競爭

### 2. 心法：你唯一的武器是「隨時走人」
- 你越像不需要這台車，車就越便宜
- 沉默的力量：報完價就閉嘴，讓對方先開口

### 3. 三句話流程（核心，可複製）
1. 報一個低到讓業務皺眉的數字 → 然後閉嘴
2. 「隔壁那家開這個價」（用競爭）
3. 「那我再想想」→ 真的站起來走向門口

### 4. 拆解車行話術與陷阱
- 「這是今天限定」「經理不會答應」黑白臉
- 灌水項目：手續費、鍍膜、延保、配件 → 逐項砍
- Trade-in 分開談，別讓它混淆新車價

### 5. 幫別人殺價（進階：你幫同事砍 $5000 的版本）
- 當第三方為什麼更好殺
- 情緒抽離、扮黑臉的技巧

### 6. 收尾與檢查表
- 簽約前最後 checklist（逐項對數字）
- 一頁可列印的「殺價作戰卡」
